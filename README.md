# カッパ係数計算アプリ (Shiny)

2人の評価者によるデータから、一致率、期待一致率、およびカッパ係数を計算するShinyアプリです。

#　開発について

本アプリのコードの大部分は、生成AI（LLM）を用いた対話とサポートによって作成・最適化されています。
